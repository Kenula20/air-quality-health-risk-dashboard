# Air Quality Health Risk Assessment Dashboard

This project is an interactive Streamlit dashboard developed for the **5DATA004C Data Science Project Lifecycle** coursework.

## Project Overview
The dashboard analyses an air quality health risk assessment dataset and provides interactive visualisations to help users explore patterns in air pollution, health indicators, and regional differences.

## Features
- Interactive dashboard built with Streamlit
- Multiple visual sections with dedicated filters
- Country and regional comparisons
- Pollutant-based analysis
- Searchable data table
- Cleaned dataset prepared for dashboard use

## Files in this Repository
- `app.py` - main Streamlit application
- `air_quality_health_risk_assessment_cleaned.csv` - cleaned dataset used in the dashboard
- `requirements.txt` - Python dependencies for deployment
- `README.md` - project documentation
